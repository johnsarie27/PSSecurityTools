# Get-CountryCode

## SYNOPSIS
Get Country Code

## SYNTAX

### __cde (Default)
```
Get-CountryCode [-Code] <String> [<CommonParameters>]
```

### __cty
```
Get-CountryCode [-Country] <String> [<CommonParameters>]
```

## DESCRIPTION
Get country from 2- or 3-letter country code

## EXAMPLES

### EXAMPLE 1
```
Get-CountryCode AS
Returns the country data for "American Somoa"
```
