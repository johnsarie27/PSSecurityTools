# Get-FileHeader

## SYNOPSIS
Get file header

## SYNTAX

```
Get-FileHeader [-Path] <String> [[-Bytes] <Int16>] [<CommonParameters>]
```

## DESCRIPTION
Get the desired number of bytes (starting from the beginning) of a file
(a.k.a., magic numbers)

## EXAMPLES

### EXAMPLE 1
```
Get-FileHeader -Path "C:\myFile.xlsx"
Returns the first 4 bytes as hex
```
