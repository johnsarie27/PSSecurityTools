# Get-WhoIs

## SYNOPSIS
Get WhoIs information

## SYNTAX

```
Get-WhoIs [-IPAddress] <String[]> [<CommonParameters>]
```

## DESCRIPTION
Get WhoIs information

## EXAMPLES

### EXAMPLE 1
```
Get-WhoIs -IPAddress '8.8.8.8'
Get's WhoIs info for the ip 8.8.8.8
```
