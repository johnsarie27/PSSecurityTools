# Get-Decoded

## SYNOPSIS
Decode string from Base64 or URL encoding

## SYNTAX

```
Get-Decoded [-String] <String> [[-Encoding] <String>] [<CommonParameters>]
```

## DESCRIPTION
Decode string from Base64 or URL encoding

## EXAMPLES

### EXAMPLE 1
```
Get-Decoded -String $encodedString
Decode $encodedString from Base64
```
