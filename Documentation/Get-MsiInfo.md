# Get-MsiInfo

## SYNOPSIS
Get MSI information

## SYNTAX

```
Get-MsiInfo [-Path] <FileInfo> [-Property] <String> [<CommonParameters>]
```

## DESCRIPTION
Get specific MSI property information

## EXAMPLES

### EXAMPLE 1
```
Get-MsiInfo -Path "C:\myMsi.msi" -Property ProductCode
Returns the product code for myMsi.msi
```
