# Out-MeasureResult

## SYNOPSIS
Outputs an object that shows maximum, minimum and average of an collection of
System.TimeSpan objects.

## SYNTAX

```
Out-MeasureResult [[-Measurement] <TimeSpan[]>] [<CommonParameters>]
```

## DESCRIPTION
Outputs an object that shows maximum, minimum and average of an collection of
System.TimeSpan objects.
