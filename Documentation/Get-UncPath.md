# Get-UncPath

## SYNOPSIS
Convert a relative path to a UNC path

## SYNTAX

```
Get-UncPath [-Path] <String> [[-ComputerName] <String>] [-Unix] [<CommonParameters>]
```

## DESCRIPTION
Convert a relative path to a UNC path

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-UncPath -Path C:\Users
```

Convert C:\Users to a UNC path
