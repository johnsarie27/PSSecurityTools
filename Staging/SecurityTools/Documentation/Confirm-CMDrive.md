# Confirm-CMDrive

## SYNOPSIS
Validate PSDrive

## SYNTAX

```
Confirm-CMDrive [-PSDrive] <String> [<CommonParameters>]
```

## DESCRIPTION
Validate PSDrive

## EXAMPLES

### Example 1
```powershell
PS C:\> Confirm-CMDrive -PSDrive 764
```

Returns true if PSDrive 764 exists and is of type CMSite. Returns false otherwise
